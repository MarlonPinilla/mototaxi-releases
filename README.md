# Mototaxi Colombia - Releases

Este repositorio contiene las versiones de las aplicaciones de Mototaxi Colombia para distribución directa.

## Apps Disponibles

- **Mototaxi Admin** - Panel administrativo
- **Mototaxi Pasajero** - App para usuarios
- **Mototaxi Conductor** - App para conductores

## Descargas

Las versiones más recientes están disponibles en la sección [Releases](https://github.com/MarlonPinilla/mototaxi-releases/releases).

## Sistema de Actualizaciones

Este repositorio forma parte del sistema de actualizaciones automáticas que utiliza:
- Firebase Remote Config para configuración
- GitHub Releases para hosting de APKs
- Verificación SHA256 para seguridad

## Instalación

1. Descarga el APK correspondiente
2. Habilita "Fuentes desconocidas" en Android
3. Instala el APK

⚠️ **Importante:** Solo descarga APKs desde este repositorio oficial.
